# 0x0D. C - Preprocessor

## Objectives

	. to know macros and how to use them
	. how to include guards to your header files

## Requirements

	. Editors: vi, vim, emacs
	. Ubuntu 20.04 LTS and gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
	. Betty style compliant code. It will be checked using betty-style.pl and betty-doc.pl
	. No global variables
	. No more than 5 functions per file
	. The only C standard library functions allowed are malloc, free and exit. Any use of functions like printf, puts, calloc, realloc etcâ€¦ is forbidden
	. You are allowed to use `_putchar`

	3. 2-main.c - a program that prints the name of the file it was compiled from, followed by a new line.
